# 动态列表组件

## 前言

- 原编辑器ScrollView组件缺点：

1、可视区域外的item也算drawcall

2、初始化时不支持分帧，数量一多打开瞬间卡顿

3、不支持动态增删，每次操作都要刷新整个列表

- 动态列表组件的原理：

在ScrollView控件的基础上（参数都是直接读取编辑器设置的），自行维护一个动态列表。

## 组件支持

- 分帧渲染

- 加入对象池，不在可视区域内的回收，降低drawcall

- 选择模式（支持单选、多选）

- 动态操作（动态增删，不需刷新整个列表，并且可以自定义增删动画）
